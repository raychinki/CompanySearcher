# 江苏企业查询 UWP
江苏企业查询是一款为大众提供服务的企业信息查询工具，能够实时查询江苏省内所有企业、个体工商户和农民专业合作社的基本信息及股东、变更、抽查、年报信息，支持查看与查询经营异常名录、严重违法名录及抽查检查信息。
